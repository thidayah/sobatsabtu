# Sobat Sabtu

A sports community platform (Bandung, Indonesia) for organizing weekend group activities — public event discovery/registration, plus an internal dashboard for staff to manage events, members, and attendance.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). You'll need a `.env.local` — see [docs/project-overview.md](./docs/project-overview.md#environment-variables) for the required variables.
